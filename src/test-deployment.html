🛡️ CrimeShield Deployment Test

Test your Edge Function deployment before running the main app

What This Does:

This page tests if your Supabase Edge Function is deployed correctly. Run all tests to verify your backend is ready.

Before Testing: Make sure you've deployed the Edge Function manually (see guide below if you haven't).

1. Health Check

Tests if the Edge Function is deployed and responding. Also checks if environment variables are set.

2. User Signup

Tests if user signup works correctly. Creates a test user account.

3. Database Check

Verifies that the database table exists and is accessible.

⚠️ If Tests Fail - Deploy Manually

If any tests fail, you need to deploy the Edge Function manually. Follow these steps:

  1. Copy code from: /supabase/functions/make-server-cfc8313f/index.ts
  2. Go to: Supabase Edge Functions
  3. Create/edit function named make-server-cfc8313f
  4. Paste code and click Deploy
  5. Add environment variables in Settings → Secrets:
    • SUPABASE_URL = https://lfksrwqamtfqrexoxlnp.supabase.co
    • SUPABASE_SERVICE_ROLE_KEY = Get from Settings → API
  6. Click Deploy again after adding secrets
  7. Create database table: Run SQL from setup-database.sql
